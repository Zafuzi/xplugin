# XPLUGIN TEMPLATE

This project requires qMake - Created using [this tutorial](https://forums.x-plane.org/index.php?/forums/topic/48012-how-to-setup-your-universal-x-plane-plugin-dev-environment-on-every-platform/)

## Instructions
- Clone this repo
- Edit the plugin files
- Run qmake to create the MakeFile
- Run make to create the plugin
- Move lib.xpl.so to X-Plane/Resources/plugins
- Remove the .so extension from the plugin should say lib.xpl
- Be happy!
